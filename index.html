🚀 Link Queek To Your Server

1. Invite The Bot

Click the "Invite" button above, select your server, and click Authorize.

2. Set Your Channels

Run these simple commands in your server:

/welcome #your-welcome-channel
/leave #your-logs-channel

3. Done!

All features are ready to use — no extra setup needed!

📋 Quick Commands

General

  • /help — View all commands
  • /ping — Check bot speed
  • /profile — View user info

Music

  • /play [url] — Play a song
  • /skip — Skip current track
  • /queue — See upcoming songs

Economy

  • /daily — Claim free coins
  • /balance — Check your wallet
  • /work — Earn extra coins
